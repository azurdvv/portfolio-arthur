Informations sur le projet

  • Catégorie : Application web - Barec
  • Client : Barec Automatismes
  • Date du projet : 2024
  • Environnement : NetBeans, PHP, SQL, HTML/CSS/JS, Bootstrap, jQuery, MariaDB, Trello

Contexte

Barec Automatismes est une entreprise spécialisée dans la fabrication de composants PCB. Ces composants électroniques nécessitent un stockage dans des conditions précises pour préserver leur qualité. Une application web a donc été demandée afin d’automatiser le suivi et la gestion de ces conditions.

Missions

Mission 1 : Découverte de Bootstrap
Création d’une page web avec barre de navigation et formulaire pour découvrir le framework.

Mission 2 : Base de données bd_barec
Conception de la base de données, documentation technique et scripts SQL.

Mission 3 : Initiation MVC
Projet sous NetBeans pour explorer le pattern Modèle-Vue-Contrôleur et ses bonnes pratiques.

Mission 4 : Architecture
Structuration MVC et Bootstrap : organisation des fichiers et formulaire d’authentification.

Mission 5 : Rendu de l’application
Démonstration de la fonctionnalité développée. Projet non finalisé suite au départ des collaborateurs.
