Compétences mobilisés dans le développement du site

Informations sur le projet

  • Catégorie : Développement d'une application web
  • Client : Barec Automatismes
  • Date du projet : 2024
  • Environnement de travail :
    - IDE : NetBeans
    - Langages : PHP, SQL, HTML, CSS, JavaScript
    - Framework : Bootstrap, Jquery
    - Base de données : MariaDB
    - Gestion de projet : Méthode Agile, Trello

Contexte :

Barec Automatismes est une entreprise spécialisée dans la fabrication de composants PCB. Ces composants électroniques sont soumis à des règles strictes en matière de stockage et de température pour garantir leur qualité et leur bon fonctionnement. Afin de mieux gérer leur stock et d’assurer un suivi précis des conditions de stockage, Barec Automatismes a exprimé le besoin d’une application web permettant de centraliser et d’automatiser ces tâches.

Missions réalisées dans ce contexte :

  • Prise en main de Bootstrap : Exploration et intégration du framework pour le développement du front-end.
  • Analyse du cahier des charges : Étude des besoins pour concevoir une base de données adaptée.
  • Création et structuration de la base de données : Mise en place des tables et relations nécessaires en SQL.
  • Développement de la page vitrine : Conception et implémentation d’une interface utilisateur attrayante.
  • Développement de la fonctionnalité de gestion des PCB : Mise en place de fonctionnalités spécifiques pour la gestion des composants.
  • Intégration du modèle MVC : Organisation du code en séparant les responsabilités pour un projet structuré.
  • Utilisation d’outils de gestion de projet : Organisation et suivi des tâches via Trello.
  • Tests et corrections : Débogage et optimisation du code pour assurer le bon fonctionnement de l’application.
  • Documentation du projet : Rédaction d’un guide d’utilisation pour faciliter la prise en main de l’application.
