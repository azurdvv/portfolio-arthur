Informations sur le projet

  • Client : Syndicat des Eaux de Gévaudan (SEG)
  • Durée : 4 sprints (6 semaines)
  • Technos : Java, Android SDK, SQLite, Webservices REST
  • Outils : Android Studio, Postman, Git

Contexte

Le SEG souhaitait remplacer les relevés Excel par une application mobile Android pour simplifier la saisie sur le terrain.

L'appli embarque une base SQLite et communique avec l’API REST déployée sous GlassFish.

Missions réalisées

  • Sprint 1 : Mise en place des WebServices REST et de la librairie métier.
  • Sprint 2 : Déploiement de l’API sous GlassFish.
  • Sprint 3 : Création de l’application Android « appli_gest_vannes ».
  • Sprint 4 : Tests finaux et packaging (APK).
