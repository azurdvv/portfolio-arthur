Informations sur le projet

  • Catégorie : Application web - Framework CodeIgniter
  • Client : SOS Informatique
  • Date du projet : 2024
  • Environnement : Netbeans, CodeIgniter, PHP, SQL, HTML, CSS, Bootstrap, jQuery, MariaDB, Jira, Bitbucket

Contexte

Dans le cadre d’un projet personnel, j’ai développé une application web en PHP à destination de l’entreprise de mon père, SOS Informatique. Cette application a été conçue pour faciliter la gestion du renouvellement des antivirus proposés par l’entreprise, qui sont à reconduire chaque année. L’outil permet d’identifier les clients dont la licence arrive à expiration et de leur envoyer automatiquement une notification pour leur proposer un renouvellement.
