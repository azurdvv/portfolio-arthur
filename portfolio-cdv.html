Informations sur le projet

  • Catégorie : Application web - Concours viticole
  • Client : Union des Œnologues du Val de Loire
  • Date du projet : 2024
  • Environnement : Netbeans, CodeIgniter, PHP, SQL, HTML, CSS, JS, Bootstrap, jQuery, MariaDB, Jira, Bitbucket

Contexte

Dans le cadre d’un concours organisé par l’Union des Œnologues du Val de Loire, les producteurs de vin sont évalués selon la qualité gustative de leur production. Pour améliorer la gestion de l’événement, une application web a été développée afin de permettre aux jurés de saisir leurs notes et de consulter les résultats en ligne.

Missions

Mission 1 : Découverte du framework CodeIgniter via un projet d’initiation avec gestion d’accès.
Cette étape a permis de découvrir un framework Php avec une Architecture spéciale et complète.

Mission 2 : Développement d’un module de création du concours.
Implémentation du formulaire et des validations pour gérer les données du concours.

Mission 3 : Mise en place d’un tableau de notes et d’une fonctionnalité de clôture du concours.
Un tableau a été créé pour répertorier les notations réçuent par un vin + ajout d'un module pour clore le concours.

Mission 4 : Création d’un palmarès affichant les meilleurs vins selon leur moyenne.
Les calculs de moyenne ont été automatisés pour générer le palmarès

Mission 5 : Déploiement de l’application sur un serveur local pour démonstration.
Le déploiement a validé la configuration du serveur et la stabilité de l’environnement de test.
